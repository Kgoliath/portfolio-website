# Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript. This site showcases AI/ML projects, skills, and provides contact information.

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Clean Theme**: Modern, professional appearance
- **Project Gallery**: Display of AI/ML projects with links to GitHub and demos
- **Skills Section**: Grid layout of technical skills
- **Contact Form**: Simple form for inquiries (uses mailto)

## Project Structure

```
portfolio-website/
├── index.html         # Home page
├── about.html         # About me section
├── projects.html      # Projects showcase
├── contact.html       # Contact and resume download
├── assets/
│   ├── css/
│   │   └── style.css  # Stylesheet
│   ├── js/
│   │   └── script.js  # JavaScript for interactivity
│   └── images/        # (Add your images here)
└── README.md
```

## Local Development

To view the website locally:

1. Clone or download the repository
2. Open `index.html` in your web browser

No server required as it's a static site.
